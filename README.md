# YouTube Resources

This repository contains the commands, scripts, configuration files, and additional resources for my YouTube videos about Linux, open-source software, troubleshooting and related topics.

## Repository Structure

Each folder corresponds to one YouTube video and contains:

- 📄 Step-by-step guide
- 💻 Commands
- 📜 Scripts (if applicable)
- ⚙️ Configuration files (if applicable)
